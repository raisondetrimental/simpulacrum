## Interaction Features

### 1. Auto-Save Draft
- Automatically saves draft every 30 seconds
- Prevents data loss if browser crashes
- Shows "Draft saved at HH:MM" indicator

### 2. Quick Toggle Preferences
- Click Y or N to toggle
- Highlight changed values in yellow
- Show "unsaved changes" indicator

### 3. Smart Currency Formatting
- Auto-formats investment amounts with commas
- Supports currency selection (USD, EUR, GBP, etc.)
- Converts between currencies (optional feature)

### 4. Date Picker
- Click calendar icon to select follow-up date
- Suggests common intervals (1 week, 2 weeks, 1 month, 3 months)
- Shows days until follow-up

### 5. Meeting History Timeline
- Collapsible section
- Click to expand individual meetings
- Export to PDF option

### 6. Validation
- Required fields highlighted if empty
- Email format validation
- Investment min < max validation
- Phone number format suggestions
